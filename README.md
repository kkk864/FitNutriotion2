# FitNutrition — интернет-магазин спортивного питания

Учебный проект по брифу: многостраничный сайт магазина спортивного питания с функциями, похожими на маркетплейс.

## Соответствие брифу

- **Название:** FitNutrition — магазин спортивного питания
- **Страницы (5–7+):** Главная, Каталог, Карточка товара, О нас, Доставка и оплата, Контакты, Корзина, Личный кабинет, Блог, Акции
- **Компоненты:** каталог с фильтрацией и сортировкой, корзина и оформление заказа (демо), личный кабинет (демо-вход), отзывы/рейтинг на карточке товара, блог, акции, обратная связь (форма)
- **Дизайн:** тёмная тема (#1A1A1A, #2D2D2D), акцент оранжевый (#FF6B35), шрифты Montserrat и Roboto, адаптивная вёрстка
- **Технологии:** HTML5, CSS3, ванильный JavaScript, без бэкенда
- **Ограничения:** оплата не реальная — имитация оформления заказа; данные товаров условные

## Как запустить локально

1. Откройте папку `fitnutrition` в проводнике.
2. Запустите любой локальный сервер из корня папки, например:
   - **Python:** `python -m http.server 8080`
   - **Node (npx):** `npx serve -l 8080`
3. Откройте в браузере: `http://localhost:8080`

Либо откройте файл `index.html` напрямую в браузере (часть функций может зависеть от доступа к файлам).

## Функции

- **Гости:** просмотр каталога, статей, доставки; добавление в корзину (без обязательной регистрации для корзины)
- **Оформление заказа:** демо — заполнение формы и «подтверждение» без реальной оплаты
- **Личный кабинет:** демо-вход по имени/email (сохранение в localStorage), блок «История заказов»
- **Каталог:** фильтр по категориям и цене, сортировка (цена, название, рейтинг), поиск по `?q=`
- **Корзина:** добавление/удаление, изменение количества, счётчик в шапке
